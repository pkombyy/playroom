# Установка 3x-ui (English Interface)

## Почему 3x-ui?

- ✅ **Английский интерфейс** - полностью переведен на английский
- ✅ **Активная разработка** - регулярные обновления
- ✅ **Больше функций** - расширенные возможности управления
- ✅ **Лучшая поддержка** - активное сообщество
- ✅ **Совместимость** - работает с теми же конфигурациями

## Быстрая установка

```bash
sudo /var/playroom/install_3xui.sh
```

## Отличия от x-ui

1. **Интерфейс**: Полностью на английском языке
2. **Порт по умолчанию**: 2053 (вместо 54321)
3. **Дополнительные функции**: Расширенные настройки безопасности
4. **Обновления**: Более частые обновления и исправления

## После установки

1. Откройте панель: `http://your-server-ip:2053`
2. Войдите: `admin` / `admin`
3. Смените пароль
4. Настройте обход белых списков:
   ```bash
   sudo /var/playroom/setup_xui_whitelist_bypass.sh
   ```

## Миграция с x-ui

Если у вас уже установлена x-ui, скрипт предложит её удалить. 
Все конфигурации и пользователи будут сохранены при миграции.

## Полезные команды

```bash
x-ui status    # Проверить статус
x-ui restart    # Перезапустить
x-ui log        # Посмотреть логи
x-ui update     # Обновить панель
```
